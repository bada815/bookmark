## 💡 Markdown 사용법 핵심

| 요소 | Markdown 문법 | 렌더링 결과 |
| :--- | :--- | :--- |
| **제목** | `# 제목 1`, `## 제목 2` | 큰 제목, 중간 제목 |
| **굵게** | `**볼드 텍스트**` | **볼드 텍스트** |
| **링크** | `[링크 이름](URL 주소)` | [링크 이름](URL 주소) |
| **목록** | `* 아이템 1`, `1. 아이템 1` | 불릿 목록, 번호 목록 |
